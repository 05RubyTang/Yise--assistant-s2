## 🎯 双属性精灵处理

### 第一属性与第二属性的关系

根据游戏规则（4.23后更新）：
- **刷取时**：只计第一属性池进度
- **出货时**：第一属性和第二属性池都可能出货

### S2 双属性精灵归属

| 精灵名 | 第一属性 | 第二属性 | 主属性池 | 第二属性池 |
|--------|---------|---------|---------|-----------|
| 恶魔叮 | 恶 | 翼 | evil ✓ | wing (无方案) |
| 灵狐 | 火 | 冰 | fire ✓ | ice ✓ |
| 幽影树 | 幽 | 草 | ghost ✓ | grass ✓ |
| 音碟吼 | 普通 | 机械 | normal ✓ | mech ✓ |
| 烟花伯爵 | 火 | 毒 | fire ✓ | poison ✓ |
| 炫光迪迪 | 光 | 电 | light ✓ | electric ✓ |
| 加油海葵 | 水 | 萌 | water ✓ | cute ✓ |
| 爆焰喷喷 | 火 | 龙 | fire ✓ | dragon (无方案) |

**处理策略**：
1. 精灵归入**第一属性**的属性方案 shinies（主池）
2. 如果**第二属性**也有对应的属性方案，同时归入该方案 shinies（扩展出货范围）

## 🎮 用户使用场景

### 场景1：刷取特定精灵

**需求**：想要获得「小独角兽」

**方案选择**：
1. **单刷方案** (推荐)：`s2_single_unicorn` - 70次保底，家族池
2. **属性方案**：光系方案 (light) - 80次保底，可能出其他光系精灵

### 场景2：刷取属性池

**需求**：想要获得萌系精灵

**方案选择**：
- **S2 萌系方案** (cute)：可出 菊花梨、加油海葵、治愈兔、大耳帽兜
- 80次保底出其中任意一个

### 场景3：双属性精灵

**需求**：想要获得「灵狐」（火+冰）

**方案选择**：
1. **单刷方案** (推荐)：`s2_single_spirit_fox` - 70次保底
2. **火系方案** (fire)：主属性池，80次保底
3. **冰系方案** (ice)：第二属性池，80次保底
