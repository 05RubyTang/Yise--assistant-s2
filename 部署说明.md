## 🐛 常见问题

### 1. 推送失败：认证错误

**错误信息**：`could not read Username for 'https://github.com'`

**解决方案**：
```bash
# 使用 SSH 方式（推荐）
git remote set-url s2-test git@github.com:05RubyTang/Yise--assistant-s2.git

# 或配置 Personal Access Token
git config --global credential.helper store
```

### 2. 页面 404

**可能原因**：
- GitHub Pages 未启用
- Base path 配置错误
- 构建产物未正确推送

**检查步骤**：
1. 确认 gh-pages 分支存在
2. 确认 GitHub Pages 已配置
3. 检查 index.html 中资源路径

### 3. 图片无法加载

**可能原因**：Base path 不匹配

**解决方案**：
```bash
# 重新构建（确保 base path 正确）
VITE_BASE_PATH=/Yise--assistant-s2/ npm run build
```
